# Investigación y Justificación del DOM (Domain Object Model)

## 1. ¿Qué es el DOM en este proyecto?

El Modelo de Objetos del Dominio representa las entidades principales del problema que el sistema busca resolver y las relaciones entre ellas.

En este proyecto, el DOM modela la estructura lógica de la aplicación para la gestión de conflictos familiares, definiendo los objetos del mundo real (usuarios, grupos familiares, desacuerdos, propuestas, etc.) y cómo interactúan entre sí.

El objetivo del DOM es:

- Representar el problema real de forma estructurada.
- Servir como base para el diseño orientado a objetos.
- Facilitar la implementación bajo principios SOLID.
- Garantizar coherencia entre requisitos funcionales y modelo de clases.
